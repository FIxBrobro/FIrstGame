# 🎮 Первый платформер на Pygame

Небольшая 2D‑игра‑платформер, написанная на Python с использованием библиотеки [Pygame](https://www.pygame.org/).  
Проект создан как учебный — для тренировки работы с игровым циклом, обработкой событий, столкновениями и тайлами. 🧠

## ✨ Возможности

- 🎹 Управление персонажем с клавиатуры (движение влево/вправо, прыжок).
- 🧱 Плиточная карта уровня (платформы, стены, ямы).
- ⚠️ Обработка столкновений с платформами.
- 🪂 Простая система гравитации и прыжка.
- ⭐ Подсчёт очков (например, за собранные предметы или пройденное расстояние).
- 🏁 Экран победы/проигрыша.

## 📦 Требования

- 🐍 Python 3.10+
- 🎨 Pygame

Установить зависимости:
pip install pygame

Склонируйте репозиторий и запустите главный файл игры:

git clone https://github.com/<your-username>/<your-repo>.git
cd <your-repo>
python main.py

text

(Название файла можно изменить на своё — например, `game.py`.)

## 🕹 Управление

По умолчанию:

- `←` / `→` или `A` / `D` — движение.
- `Space` или `W` / `↑` — прыжок.
- `Esc` — выход из игры.

## 🗂 Структура проекта

Примерная структура:
├── main.py # Точка входа, игровой цикл
├── level.py # Загрузка и отрисовка уровня, тайловая карта
